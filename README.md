Project Overview : This project analyzes Aadhaar biometric authentication usage data using Power BI. The dashboard helps monitor biometric authentication trends and supports decision-making by identifying normal and abnormal usage patterns.

Problem Statement : Aadhaar biometric authentication systems generate large volumes of data daily. Manual monitoring is difficult, so there is a need for a visual system that clearly shows usage trends and highlights abnormal behavior.

Objectives : • Analyze biometric authentication usage data • Monitor trends over time • Identify normal and abnormal usage patterns • Support decision-making using visual dashboards

Tools & Technologies : • Power BI Desktop • Aadhaar public dataset (data.gov.in) • GitHub.

Dataset Description : The dataset contains biometric authentication usage information such as: • Date • State and district • Biometric usage counts by age group No personal Aadhaar information is used.

Dashboard Pages : Page 1: Biometric Usage Monitoring • Total biometric usage over time • Age-wise usage comparison • State and district-level summary Page 2: Decision Support • Overall usage status (Normal / Abnormal) • Tabular view for quick decision-making

How to Use :

Download the PBIX file from this repository
Open it using Power BI Desktop
Interact with the visuals and filters
Key Outcome : The dashboard provides a clear overview of Aadhaar biometric usage and helps authorities quickly understand system behavior and take informed decisions.
